**attio_client.py — Add objects methods:**

Attio Objects API:
- `GET /v2/objects` — list all objects
- `GET /v2/objects/{object_id_or_slug}` — get object
- `POST /v2/objects` — create object. Body: `{"data": {"api_slug": "custom_thing", "singular_noun": "Thing", "plural_noun": "Things"}}`.
- `PATCH /v2/objects/{object_id_or_slug}` — update object
- `GET /v2/objects/{object_id_or_slug}/views` — list views

NOTE: No delete endpoint for objects. This is intentional.

```python
# ── Objects operations ────────────────────────────────────────────────
def list_objects(self) -> dict[str, Any]:
    return self._request("GET", "/objects")

def get_object(self, object_id: str) -> dict[str, Any]:
    return self._request("GET", f"/objects/{object_id}")

def create_object(self, api_slug: str, singular_noun: str, plural_noun: str) -> dict[str, Any]:
    return self._request("POST", "/objects", json={"data": {"api_slug": api_slug, "singular_noun": singular_noun, "plural_noun": plural_noun}})

def update_object(self, object_id: str, data: dict[str, Any]) -> dict[str, Any]:
    return self._request("PATCH", f"/objects/{object_id}", json={"data": data})

def list_object_views(self, object_id: str) -> dict[str, Any]:
    return self._request("GET", f"/objects/{object_id}/views")
```

**attio_client.py — Add attributes methods:**

Attio Attributes API:
- `GET /v2/objects/{object}/attributes` — list attributes on object. Also works with `GET /v2/lists/{list}/attributes`.
- `GET /v2/objects/{object}/attributes/{attribute}` — get attribute
- `POST /v2/objects/{object}/attributes` — create attribute. Body: `{"data": {"title": "...", "api_slug": "...", "type": "text"}}`.
- `PATCH /v2/objects/{object}/attributes/{attribute}` — update attribute
- `POST /v2/objects/{object}/attributes/{attribute}/archive` — ARCHIVE (NOT DELETE). This is critical.
- `GET /v2/objects/{object}/attributes/{attribute}/options` — list select options
- `POST /v2/objects/{object}/attributes/{attribute}/options` — create select option
- `GET /v2/objects/{object}/attributes/{attribute}/statuses` — list status values
- `POST /v2/objects/{object}/attributes/{attribute}/statuses` — create status value

```python
# ── Attributes operations ─────────────────────────────────────────────
def list_attributes(self, target: str, target_type: str = "objects") -> dict[str, Any]:
    """target_type is 'objects' or 'lists'. target is slug or ID."""
    return self._request("GET", f"/{target_type}/{target}/attributes")

def get_attribute(self, object_slug: str, attribute_slug: str) -> dict[str, Any]:
    return self._request("GET", f"/objects/{object_slug}/attributes/{attribute_slug}")

def create_attribute(self, object_slug: str, title: str, api_slug: str, type: str) -> dict[str, Any]:
    return self._request("POST", f"/objects/{object_slug}/attributes", json={"data": {"title": title, "api_slug": api_slug, "type": type}})

def update_attribute(self, object_slug: str, attribute_slug: str, data: dict[str, Any]) -> dict[str, Any]:
    return self._request("PATCH", f"/objects/{object_slug}/attributes/{attribute_slug}", json={"data": data})

def archive_attribute(self, object_slug: str, attribute_slug: str) -> dict[str, Any]:
    return self._request("POST", f"/objects/{object_slug}/attributes/{attribute_slug}/archive")

def list_attribute_options(self, object_slug: str, attribute_slug: str) -> dict[str, Any]:
    return self._request("GET", f"/objects/{object_slug}/attributes/{attribute_slug}/options")

def create_attribute_option(self, object_slug: str, attribute_slug: str, title: str) -> dict[str, Any]:
    return self._request("POST", f"/objects/{object_slug}/attributes/{attribute_slug}/options", json={"data": {"title": title}})

def list_attribute_statuses(self, object_slug: str, attribute_slug: str) -> dict[str, Any]:
    return self._request("GET", f"/objects/{object_slug}/attributes/{attribute_slug}/statuses")

def create_attribute_status(self, object_slug: str, attribute_slug: str, title: str) -> dict[str, Any]:
    return self._request("POST", f"/objects/{object_slug}/attributes/{attribute_slug}/statuses", json={"data": {"title": title}})
```

- `get` — arguments: `object_slug`, `attribute_slug`, option: `--json`
- `create` — options: `--object` (required), `--title` (required), `--slug` (required), `--type` (required, e.g. "text", "number", "select"), `--json`
- `update` — arguments: `object_slug`, `attribute_slug`, options: `--data` (required, JSON), `--json`
- `archive` — arguments: `object_slug`, `attribute_slug`, options: `--yes`, `--json`. Calls `client.archive_attribute()`. MUST confirm before archiving. CRITICAL: this is ARCHIVE, not delete.
- `options` — arguments: `object_slug`, `attribute_slug`, option: `--json`. Lists select options. Subcommand for listing.
- `options-create` — arguments: `object_slug`, `attribute_slug`, option: `--title` (required), `--json`. Creates a select option.
- `statuses` — arguments: `object_slug`, `attribute_slug`, option: `--json`. Lists status values.
- `statuses-create` — arguments: `object_slug`, `attribute_slug`, option: `--title` (required), `--json`. Creates a status value.
