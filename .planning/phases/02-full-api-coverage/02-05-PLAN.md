**attio_client.py — Add files methods:**

Attio Files API:
- `POST /v2/files/upload` — upload file. Multipart form: `file` (binary), `record_id` (string), `object` (string). Returns file metadata.
- `GET /v2/files/{file_id}` — get file metadata
- `GET /v2/files` — list files. Query params: `record_id`, `object`.
- `GET /v2/files/{file_id}/download` — download file binary. Returns raw bytes (NOT JSON).
- `DELETE /v2/files/{file_id}` — delete file
- `POST /v2/files/folders` — create folder. Body: `{"data": {"name": "...", "record_id": "...", "object": "..."}}`.

CRITICAL: `upload_file` and `download_file` need special handling since they don't follow the standard `_request` -> JSON pattern.

```python
# ── Files operations ──────────────────────────────────────────────────
def upload_file(self, file_path: str, record_id: str, object_slug: str) -> dict[str, Any]:
    """Upload binary file via multipart form. Does NOT use _request (different content type)."""
    with open(file_path, "rb") as f:
        resp = self._client.post(
            "/files/upload",
            files={"file": (Path(file_path).name, f)},
            data={"record_id": record_id, "object": object_slug},
        )
    if resp.status_code == 401:
        raise AuthError()
    if resp.status_code == 404:
        raise NotFoundError(f"record {record_id}")
    resp.raise_for_status()
    return resp.json()

def get_file_info(self, file_id: str) -> dict[str, Any]:
    return self._request("GET", f"/files/{file_id}")

def list_files(self, record_id: str | None = None, object_slug: str | None = None) -> dict[str, Any]:
    params: dict[str, Any] = {}
    if record_id: params["record_id"] = record_id
    if object_slug: params["object"] = object_slug
    return self._request("GET", "/files", params=params)

def download_file(self, file_id: str) -> bytes:
    """Download file binary. Returns raw bytes (NOT JSON)."""
    resp = self._client.get(f"/files/{file_id}/download")
    if resp.status_code == 401:
        raise AuthError()
    if resp.status_code == 404:
        raise NotFoundError(f"file {file_id}")
    resp.raise_for_status()
    return resp.content

def delete_file(self, file_id: str) -> dict[str, Any]:
    return self._request("DELETE", f"/files/{file_id}")

def create_folder(self, name: str, record_id: str, object_slug: str) -> dict[str, Any]:
    return self._request("POST", "/files/folders", json={"data": {"name": name, "record_id": record_id, "object": object_slug}})
```

**attio_client.py — Add meetings methods:**

Attio Meetings API (read-only):
- `GET /v2/meetings` — list meetings. Returns `{"data": [...]}`.
- `GET /v2/meetings/{meeting_id}` — get meeting
- `GET /v2/meetings/{meeting_id}/recordings` — list recordings for a meeting
- `GET /v2/meetings/{meeting_id}/recordings/{recording_id}/transcript` — get transcript

```python
# ── Meetings operations (read-only) ──────────────────────────────────
def list_meetings(self) -> dict[str, Any]:
    return self._request("GET", "/meetings")

def get_meeting(self, meeting_id: str) -> dict[str, Any]:
    return self._request("GET", f"/meetings/{meeting_id}")

def list_meeting_recordings(self, meeting_id: str) -> dict[str, Any]:
    return self._request("GET", f"/meetings/{meeting_id}/recordings")

def get_meeting_transcript(self, meeting_id: str, recording_id: str) -> dict[str, Any]:
    return self._request("GET", f"/meetings/{meeting_id}/recordings/{recording_id}/transcript")
```

**files.py — Command group:**

`files_group = click.Group("files", help="Upload, download, and manage files on records.")`:
- `upload` — options: `--file` (required, type=click.Path(exists=True)), `--record-id` (required), `--object` (required), `--json`. Calls `client.upload_file()`.
- `get` — argument: `file_id`, option: `--json`. Calls `client.get_file_info()`.
- `list` — options: `--record-id`, `--object`, `--json`. Calls `client.list_files()`.
- `download` — argument: `file_id`, option: `--output` (output path, default=stdout). Calls `client.download_file()`. If `--output` provided, write bytes to that path and print confirmation. If not, write raw bytes to stdout (for piping).
- `delete` — argument: `file_id`, options: `--yes`, `--json`. Calls `client.delete_file()`.
- `create-folder` — options: `--name` (required), `--record-id` (required), `--object` (required), `--json`.

CRITICAL for download: When writing to stdout as binary, use `sys.stdout.buffer.write()` not `click.echo()`.
