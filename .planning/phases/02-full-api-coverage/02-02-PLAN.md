**attio_client.py — Add comments methods:**

Attio Comments API:
- `POST /v2/comments` — create comment. Body: `{"data": {"thread_id": "thrd_xxx" | null, "body": "...", "record_id": "rec_xxx", "entry_id": "ent_xxx" (optional)}}`. When `thread_id` is null and `record_id` is provided, a new thread is created.
- `GET /v2/comments/{comment_id}` — get comment
- `DELETE /v2/comments/{comment_id}` — delete comment
- `POST /v2/comments/{comment_id}/resolve` — resolve thread via comment
- `POST /v2/comments/{comment_id}/unresolve` — unresolve thread

Add methods:
```python
# ── Comments operations ───────────────────────────────────────────────
def create_comment(self, body: str, record_id: str | None = None, entry_id: str | None = None, thread_id: str | None = None) -> dict[str, Any]:
    data: dict[str, Any] = {"body": body}
    if thread_id: data["thread_id"] = thread_id
    if record_id: data["record_id"] = record_id
    if entry_id: data["entry_id"] = entry_id
    return self._request("POST", "/comments", json={"data": data})

def get_comment(self, comment_id: str) -> dict[str, Any]:
    return self._request("GET", f"/comments/{comment_id}")

def delete_comment(self, comment_id: str) -> dict[str, Any]:
    return self._request("DELETE", f"/comments/{comment_id}")

def resolve_comment(self, comment_id: str) -> dict[str, Any]:
    return self._request("POST", f"/comments/{comment_id}/resolve")

def unresolve_comment(self, comment_id: str) -> dict[str, Any]:
    return self._request("POST", f"/comments/{comment_id}/unresolve")
```

**attio_client.py — Add threads methods:**

Attio Threads API:
- `GET /v2/threads` — list threads. Query params: `record_id`, `entry_id`.
- `GET /v2/threads/{thread_id}` — get thread with all comments

```python
# ── Threads operations ────────────────────────────────────────────────
def list_threads(self, record_id: str | None = None, entry_id: str | None = None) -> dict[str, Any]:
    params: dict[str, Any] = {}
    if record_id: params["record_id"] = record_id
    if entry_id: params["entry_id"] = entry_id
    return self._request("GET", "/threads", params=params)

def get_thread(self, thread_id: str) -> dict[str, Any]:
    return self._request("GET", f"/threads/{thread_id}")
```

**comments.py — Create two command groups in one file:**

`comments_group = click.Group("comments", help="Manage comments on threads/records.")`:
- `create` — options: `--body` (required), `--record-id`, `--entry-id`, `--thread-id`, `--json`. At least one of `--record-id` or `--thread-id` must be provided (validate in command).
- `get` — argument: `comment_id`, option: `--json`
- `delete` — argument: `comment_id`, options: `--yes`, `--json`
- `resolve` — argument: `comment_id`, option: `--json`. Calls `client.resolve_comment()`.
- `unresolve` — argument: `comment_id`, option: `--json`. Calls `client.unresolve_comment()`.

`threads_group = click.Group("threads", help="Manage comment threads.")`:
- `list` — options: `--record-id`, `--entry-id`, `--json`
- `get` — argument: `thread_id`, option: `--json`
